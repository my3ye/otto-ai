NS.com = Network School, a Balaji Srinivasan-affiliated residential startup society based in Singapore. NOT a VC fund.

**Cost:** $1,500-$3,000/month residential membership
**Founder:** Balaji Srinivasan (Network State book author, former Coinbase CTO, a16z GP)
**Mission:** "Turn internet communities into physical startup societies"

### Three Tracks
- **Learn:** Blockchain, AI, distributed systems, startup societies curriculum. 56-week reading list (Balaji, Vitalik, Thiel, Ammous). Blockchain-verified credentials.
- **Earn:** Bounties paid in USDC/ETH/SOL, $500-$20K/task. Dev, design, content, QA categories.
- **Burn:** Fitness programming.

### Application
- Free, rolling basis, 2-3 week response
- Teams apply together (individual submissions)
- Singapore visa support available
- Looks for: builders, entrepreneurs, developers, designers

### MY3YE Alignment: EXCEPTIONAL
Balaji's Network State thesis maps almost exactly to MY3YE:
- Tusita = physical startup society node
- ONEON = sovereign network
- SOS Systems = refuge + education ladder
- Otto = decentralized intelligence

NOT a fundraising target. It's a community to join and operate within.
1. Apply as a team — lead with Tusita as parallel civilization / physical network state node
2. Reference The Network State book to show philosophical depth
3. Use Earn platform to post MY3YE bounties — access Web3-native builders
4. Network State Conference = right venue for PiPi/Polkadot pitch

**Why:** $1,500-$3,000/month residency cost, not fundraising
