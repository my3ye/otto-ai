## Part 2: Automatic Token-Holding Eligibility System Design

### Core Concept
A wallet connects once. The system reads their cross-chain holdings, calculates a **Community Tier** and an **Alignment Score**. Together these determine their perks in the MY3YE ecosystem.

---

```
Category A: MISSION ALIGNED (public goods, sovereignty, infrastructure)
  EVM: GTC (Gitcoin), OP (Optimism), $ENS, ARB (Arbitrum)
  Threshold: hold any amount

Category B: CULTURE ALIGNED (meme culture, community-driven, social)
  Solana: BONK, WIF, POPCAT
  EVM/Base: DEGEN (Farcaster)
  EVM: BANK (BanklessDAO), FWB
  Threshold: hold any amount (or minimum dust threshold to prevent farming)

  EVM: NOUN (Nouns DAO NFT), UNI, veOP (locked OP)

Category D: ECOSYSTEM NATIVE (MY3YE own tokens)
  Solana: $KOIN, $KOINK
  EVM: $KOIN (chain agnostic)
  Threshold: hold any amount → automatic boost to Tier 2+
```

### Alignment Score System (0–100)

The Alignment Score measures how deeply someone is committed to the values MY3YE is building toward. Published on-chain via attestation for transparency (Mev requirement).

**Scoring Factors (total = 100 points):**

```
A. TOKEN DIVERSITY (25 pts)
   - Hold 1 category: 5 pts
   - Hold 2 categories: 10 pts
   - Hold 3 categories: 18 pts
   - Hold all 4 categories: 25 pts

B. HOLDING CONVICTION (20 pts)
   - Holding duration >30 days: 5 pts
   - Holding duration >90 days: 10 pts
   - Holding duration >1 year: 15 pts
   - Never sold primary position (diamond hands): +5 pts bonus
   [Measured via on-chain transfer history]

C. GOVERNANCE PARTICIPATION (20 pts)
   - Cast ≥1 on-chain vote in any listed DAO: 5 pts
   - Cast ≥5 votes: 10 pts
   - Submitted ≥1 proposal: +5 pts bonus
   - Gitcoin donor (any amount): +5 pts bonus
   [Sources: Snapshot API, on-chain vote records, Gitcoin grants API]

D. SOCIAL GRAPH ALIGNMENT (20 pts)
   - ENS name: 5 pts
   - Farcaster profile (≥30 followers): 5 pts
   - Lens Profile NFT: 5 pts
   - BrightID verified: 3 pts
   - Human Passport score ≥20: 2 pts
   [Sources: ENS registry, Farcaster API, Lens API, Gitcoin Passport API]

E. CONTRIBUTION HISTORY (15 pts)
   - Gitcoin Grants donor: 5 pts
   - POAP holder (any legit POAP): 3 pts
   - Optimism RPGF voter/contributor: 5 pts
   - DAO treasury contributor: 2 pts
   [Sources: Gitcoin API, POAP API, Snapshot, chain events]
```

**Score thresholds:**
- 0–19: Explorer
- 20–39: Participant
- 40–59: Contributor
- 60–79: Builder
- 80–100: Sovereign

### Technical Implementation

#### Stack
```
Backend:
  - Off-chain eligibility API (FastAPI, integrates with Otto memory at :8100)
  - Multi-chain RPC calls (Helius for Solana, Alchemy/Infura for EVM)
  - Attestation layer: EAS (Ethereum Attestation Service) for score publishing

Frontend:
  - Wallet connect: RainbowKit (EVM) + Solana Wallet Adapter
  - Eligibility dashboard on my3ye.xyz or otto.lk/eligibility
  - Real-time score display with breakdown

On-Chain (optional Phase 2):
  - Soulbound Token (ERC-5484) for Tier status
  - EAS Schema for Alignment Score attestation
```

#### API Flow
```
1. User connects wallet(s) — Solana + EVM (can link multiple)
2. POST /eligibility/check
   Body: { solana_wallet, evm_wallet, signature }
3. System queries in parallel:
   - Helius: BONK, WIF, $KOIN, $KOINK balances
   - Alchemy: GTC, OP, ENS, DEGEN, BANK balances
   - ENS Registry: name ownership
   - Farcaster Hub: profile + follower count
   - Gitcoin Passport: score
   - Snapshot: vote history across listed DAOs
   - On-chain vote records (optional, async)
4. Score calculated, tier assigned
5. Response: { tier, alignment_score, breakdown, perks }
6. Optional: POST signed attestation to EAS with score
```

#### Re-verification
- Eligibility cached with 24h TTL
- On every action (airdrop claim, governance vote), re-verify before applying multiplier
- "Snapshot" approach for airdrops: score at a specific block height

### Anti-Farming Safeguards
- **Minimum hold thresholds**: For BONK/WIF, require ≥$5 USD equivalent (blocks dust)
- **Wallet age gate**: Wallet must be ≥30 days old (from Sybil resistance research)
- **Human Passport optional gate**: Score ≥20 for Tier 3+ (from existing research)
- **World ID nullifier**: For Tier 4 sovereign status — proof of unique humanity
- **No reward for buying day-of**: Conviction score requires hold history → pre-existing holders benefit most (natural ambassador effect)
