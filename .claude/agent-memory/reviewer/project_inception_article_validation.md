Validated synthesis for Paragraph + X thread inception article on 2026-03-22. VALIDATION_SCORE: 7/10.

**Critical finding:** Synthesis Insight #3 claims "8 projects are GREEN/live" (Otto AI, MY3YE, 505 Systems, ONEON, Panik, Shakrah, Tusita, WebAssist). But `universe/registry.yaml` status legend is authoritative:
- `live` = deployed, users can interact → only Assistive Tech (WebAssist)
- `active` = being built, has code/content → MY3YE, Otto AI
- `early` = repo exists, shell/placeholder → ONEON, Tusita, 505 Systems, Shakrah, Panik
- `concept` → all platforms, Ottolabs, Koink, PiPi

BRANDS_AND_PROJECTS.md confirms: 505.systems has "no live site"; Shakrah has "Domain: None yet."

**Why:** Synthesis pulled from an "ecosystem audit" that used its own 83-100% completeness metric — not the registry's live/early/concept taxonomy. Conflating these in a public Web3 article = credibility risk.

**How to apply:** When reviewing any content synthesis for this ecosystem, always cross-check project status claims against `universe/registry.yaml` status legend. "Early" projects have landing pages/shells — not live products. Distinguish "domain registered" from "functional site" from "live product."

Recommended content-creator correction: claim 3 fully live + 5 early/active projects with presence, rather than "8 live."
