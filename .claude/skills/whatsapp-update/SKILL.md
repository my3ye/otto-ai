## Message Guidelines

- Short, clear, direct — like a co-founder texting their co-founder
- Lead with the point, not the context
- Every 3rd message should include a question
- Don't just report — collaborate, propose, ask

## Message Types (pick one)

1. **Decision request**: "I'm considering X. Two approaches: [A] vs [B]. I lean toward [A] because [reason]. Thoughts?"
2. **Progress + next step**: "Finished X. Planning Y next. Sound right?"
3. **Blocker + ask**: "Stuck on X. Tried Y and Z. Direction?"
4. **Status + insight**: "Completed X. Learned [thing]. No questions right now."

Types 1-3 should be the majority.

## Skip messaging only when

- Literally nothing happened
- Substantive update sent last cycle with no changes since
