### Architecture Decision
Add a **view toggle (List | Month | Week)** to both character calendar pages.
- Keep existing list view intact — just add calendar as an alternate render mode
- New shared component: `src/components/social-calendar/CalendarView.tsx`
- Monthly: 7-column grid (Sun-Sat), each day cell shows post dots by status color
- Weekly: 7-column layout showing days in current week with ordered posts per column
- Post preview: clicking a post shows the existing edit Sheet (reuse existing state)
- Clicking empty day → creates new post with that date pre-filled
- No new API endpoints needed — existing `/social-calendar?character=X&limit=200` is sufficient
