## Content Type Definitions

| Type | Format | Rules |
|------|--------|-------|
| **Single Post** | 1 tweet, ≤280 chars | No hashtags. No emojis. Standalone — but references the narrative arc readers already know. |
| **Thread** | 2–9 numbered tweets | First tweet must hook without context. Last tweet has CTA. Links in reply only (not tweet body). |
| **Visual** | Tweet + image/card | Not scheduled for Phase 1 — visual assets TBD after Phase 2 performance data. |

---

## Content Rules

- **No hashtags.** Zero. Not even #MY3YE. The content either resonates or it doesn't — we don't chase algorithm shortcuts.
- **No emojis.** Brand rule: strict.
- **No URLs in tweet body.** X algorithm de-ranks tweets with outbound links. Put all links in reply threads.
- **No startup speak.** "exciting to share," "introducing," "revolutionary" — banned.
- **No urgency.** No "don't miss this," no countdown language. The river doesn't beg.
- **All lowercase for first tweet of each thread.** Matches existing brand voice ("the eye that sees...").
- **Capital letters for standalone declarations.** "Power is not a lake." Not "power is not a lake."
